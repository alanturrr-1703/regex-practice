What's inside

🧠

Deep reading — not a cheat sheet

Every concept has a notes.md (400–600 lines) covering the NFA engine, backtracking, Java-specific escaping, interview traps, and production concerns.

✍️

You implement everything

Every Solution.java starts with throw new UnsupportedOperationException("TODO"). No answers given — only progressive hints.

✅

Real JUnit test suite

The desktop app runs ./gradlew :concept:difficulty:problem:test — every @Test from the real SolutionTest.java is shown, not a 5-case subset.

⚡

Monaco editor (VS Code)

Full IntelliSense-quality editor with Java syntax highlighting, auto-indent, and direct disk saving — your code persists between sessions.

🔒

Fully offline

No external APIs. Code runs on your local JDK via Electron IPC. Works on a plane, in a bunker, anywhere.

🎯

Graduated difficulty

Easy → Medium → Hard within each concept. Hard problems include catastrophic backtracking repair, ReDoS-safe tokenizers, and production log parsers.
